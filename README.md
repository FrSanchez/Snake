# snake
Traditional game of snake for cocos2dx

# Compiling
Only ios-build is known to work
```
cd ios-build
open snake-mobile.xcodeproj
```

# Game Assets:
* Snake: https://opengameart.org/content/snake-graphics
* Sounds: https://soundimage.org/attribution-info/
